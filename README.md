# Luxury Buddhist Indian Wedding Invitation

A grand, high-end "Raabta" inspired interactive wedding invitation. This project combines modern web animations with the rich cultural aesthetic of a North Indian Buddhist wedding.

## Features
- **Raabta Scroll Logic**: A dynamic SVG line "connects" sections as you scroll.
- **Buddhist Aesthetic**: Features dharma wheels, lotuses, and serene Buddha imagery in vibrant luxury pastels.
- **Scroll Parallax Procession**: A moving Baraat (wedding procession) that enters from the sides using scroll-triggered transforms.
- **Sticky Countdown**: Gold-themed timer for the wedding Muhurat.
- **Interactive Audio**: Minimalist player controls a Buddhist-Indian fusion background track (triggers upon interaction).
- **Luxury RSVP**: A glassmorphism form with confetti celebration effects on submission.

## Tech Stack
- **React.js**: Front-end framework.
- **Framer Motion**: Advanced scroll-based animations.
- **Tailwind CSS**: Luxury theme styling (Gold, Peach, Lavender, Mint).
- **Lucide React**: Iconography.
- **Canvas-Confetti**: Interactive particle effects.

## Prerequisites
- Node.js (v16.0.0 or higher)
- npm or yarn
